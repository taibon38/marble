# MARBLE

MARBLE（MARVEL + ASSEMBLE)とは、MARVEL好きによる、MARVEL好きのための映画検索サイトです。

 
# Features
 
- MARVEL映画に特化した検索・解説が読めます
- 好きなキャラクター/楽しみ方で検索も可能
